# Prompt Lab (Image Prompt Generator + Prompt Tools)

## What is this?
A tool that helps users generate:
- image prompts
- structured prompts for coding or brainstorming
- variations (short/long/style)

## MVP
- User selects:
  - topic
  - style
  - mood
  - length
- Tool generates prompt text
- Copy-to-clipboard button

## Phase 2
- Save prompt history for members
- Share prompt links
- Rate limit generation (if AI-backed)

## Testing checklist
- Copy works
- Input validation works
